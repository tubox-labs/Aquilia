# I18n Content

The I18n Content example demonstrates localized application copy, `Accept-Language`
header negotiation, fallback behavior, and plural-aware translations using Aquilia's
internationalization subsystem.

---

## What It Demonstrates

- `I18nIntegration` configuration via workspace `.i18n()` method
- `I18nService` with `MemoryCatalog` for in-memory translation storage
- `I18nConfig` with locale negotiation, fallback, and missing key strategy
- `Accept-Language` header parsing for automatic locale detection
- Pluralization rules per locale (`count=0`, `count=1`, `count>1`)
- Dotted catalog key paths into nested dictionaries

## Key Files

| File | Purpose |
| ---- | ------- |
| `workspace.py` | Configures i18n with three locales and catalog directories |
| `modules/content/manifest.py` | Declares `ContentController` and `ContentLocalizationService` |
| `modules/content/controllers.py` | Localized content endpoint reading `Accept-Language` |
| `modules/content/services.py` | `ContentLocalizationService` composing `I18nService` |

## Workspace Configuration

```python
from aquilia import Module, Workspace
from aquilia.integrations import DiIntegration

workspace = (
    Workspace("i18n-content-app", version="1.0.0")
    .runtime(mode="dev", host="127.0.0.1", port=8065, reload=True)
    .module(Module("content", version="1.0.0").route_prefix("/content").tags("i18n"))
    .i18n(
        default_locale="en",
        available_locales=["en", "es", "fr"],
        catalog_dirs=["locales"],
        missing_key_strategy="return_key",
    )
    .integrate(DiIntegration(auto_wire=True))
)
```

| Setting | Purpose |
| ------- | ------- |
| `default_locale` | Fallback locale when no preference is expressed |
| `available_locales` | Supported locales for negotiation |
| `catalog_dirs` | Directories containing locale catalog files |
| `missing_key_strategy` | Behavior when a translation key is not found: `"return_key"`, `"raise"`, `"return_empty"` |

## Localization Service

The service composes `I18nService`, `I18nConfig`, and `MemoryCatalog`:

```python
from aquilia.i18n import I18nConfig, I18nService, MemoryCatalog, negotiate_locale

class ContentLocalizationService:
    def __init__(self):
        catalogs = {
            "en": MemoryCatalog({
                "landing.title": "Welcome",
                "landing.greeting": "Hello, {name}!",
                "items.count.0": "No items",
                "items.count.1": "One item",
                "items.count.many": "{count} items",
            }),
            "es": MemoryCatalog({
                "landing.title": "Bienvenido",
                "landing.greeting": "Hola, {name}!",
                "items.count.0": "Sin artículos",
                "items.count.1": "Un artículo",
                "items.count.many": "{count} artículos",
            }),
            "fr": MemoryCatalog({...}),
        }
        config = I18nConfig(
            default_locale="en",
            available_locales=["en", "es", "fr"],
            missing_key_strategy="return_key",
        )
        self._i18n = I18nService(config=config, catalogs=catalogs)

    async def translate(self, accept_language: str | None, key: str, **kwargs) -> str:
        locale = negotiate_locale(accept_language, self._i18n.config.available_locales)
        return self._i18n.translate(locale, key, **kwargs)
```

## Locale Negotiation

The `negotiate_locale()` function parses the `Accept-Language` header and selects the
best matching locale from the available set:

```python
# Header: "Accept-Language: es, en;q=0.9"
# Available: ["en", "es", "fr"]
# Result: "es"

# Header: "Accept-Language: de"
# Available: ["en", "es", "fr"]
# Result: "en" (default)
```

## Pluralization

Plural rules are encoded as dotted suffix keys:

```
items.count.0     → "No items"          (count == 0)
items.count.1     → "One item"          (count == 1)
items.count.many  → "5 items"           (count > 1)
```

```python
text = self._i18n.translate(locale, "items.count", count=3)
# Returns "3 items" / "3 artículos" / "3 articles"
```

## Catalog Keys

Keys use dotted paths corresponding to nested dictionary structure:

```python
MemoryCatalog({
    "landing.title": "Welcome",
    "landing.greeting": "Hello, {name}!",
    "errors.auth.failed": "Authentication failed",
    "errors.auth.expired": "Session expired",
})
```

## Running

```bash
cd examples/i18n_content_app
python -m uvicorn runtime:app --reload --port 8065
```

```bash
# English (default)
curl "http://127.0.0.1:8065/content/landing?name=Ana&count=3"

# Spanish
curl "http://127.0.0.1:8065/content/landing?name=Ana&count=3" \
  -H "Accept-Language: es"

# French
curl "http://127.0.0.1:8065/content/landing?name=Ana&count=3" \
  -H "Accept-Language: fr"

# Unknown locale (falls back to en)
curl "http://127.0.0.1:8065/content/landing?name=Ana&count=1" \
  -H "Accept-Language: de"

# Run tests
python -m pytest examples/i18n_content_app -q
```
