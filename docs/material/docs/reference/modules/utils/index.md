# Utils Module

> `aquilia.utils` — Utility functions and helpers

The Utils module provides miscellaneous utility functions and helper classes used internally by the Aquilia framework.

## When to Use

Use the Utils module when you need general-purpose helpers for:

- String manipulation and sanitization
- Date/time formatting
- Encoding and decoding
- Path manipulation
- Import utilities
